================================================================================
 改版履歴
================================================================================
2017/02 Sample Program for MDK_PHP ver 1.0.0 リリース

    ※以下、VeriTrans3G版MDKからの更新分を記載しています。

    クレジットカードを利用する決済共通：
      ・MDKトークンサーバーから取得したトークンを利用して決済要求を行えるように
        対応

2017/10 Sample Program for MDK_PHP ver 1.0.1 リリース

    コンビニ決済、銀行決済：
      ・決済申込時にプッシュURLを設定できるように対応

2018/02 Sample Program for MDK_PHP ver 1.0.2 リリース

    コンビニ決済：
      ・決済申込時に支払期限時分を設定できるように対応
      ・検索条件に支払期限日時を設定できるように対応
    Alipay決済：
      ・バーコード決済（店舗スキャン型）の確認（Confirm）要求に対応
    キャリア決済：
      ・auかんたん決済の部分返金に対応
